Welcome to Tsunami Simulation's documentation!
==============================================

Build: scons

Run the dam breaking: ./build/tsunami_lab

Run all tests: ./build/tests

Generate Doxygen documentation: doxygen ./Doxyfile

Generate HTML-Sphinx documentation: call "make html" inside ./sphinx

To list all available formats, use "make" without additional arguments inside ./sphinx
